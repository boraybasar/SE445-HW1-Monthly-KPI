# SE445-HW1-Monthly-KPI
SE445 HW1 - Monthly KPI Reporter workflow and implementation
## HW2 – Raw Data Export

This part of the project reads KPI values from a single sheet and sends the raw numbers in an email-friendly format.

Implemented flow:
Trigger → Read KPI Data → Format Email → Send Email

Included raw KPI values:
- Revenue
- New Customers
- Churn
- Support Tickets
